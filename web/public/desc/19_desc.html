mat_prj

Task

Create a function named mat_prj that takes a matrix as its argument and returns the projection matrix onto the column space of AAA.

Input

  • a: mat - Input matrix

Output

  • mat - Projection matrix onto the column space of AAA

Theory

The projection matrix PPP that projects vectors onto the column space of matrix AAA is given by:
P=A(ATA)−1AT P = A(A^TA)^{-1}A^T P=A(ATA)−1AT

This matrix has the property that for any vector b\mathbf{b}b, the product PbP\mathbf{b}Pb gives the projection of b\mathbf{b}b onto the column space of AAA.

Example

A=(10),P=(1000) A = \begin{pmatrix} 1 \\ 0 \end{pmatrix}, \quad P = \begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix} A=(10​),P=(10​00​)

Test

a: mat = [[1], [0]]
result = mat_prj(a)
np.testing.assert_allclose(result, [[1, 0], [0, 0]], atol=1e-10)
