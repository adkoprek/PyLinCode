ortho

Task

Create a function named ortho that takes a list of orthogonal vectors and a new vector as its arguments and returns the component of the new vector that is orthogonal to all vectors in the list using the Gram-Schmidt process.

Input

  • vecs: list[vec] - List of mutually orthogonal vectors
  • new: vec - New vector to orthogonalize

Output

  • vec - Component of new orthogonal to all vectors in vecs

Theory

The Gram-Schmidt orthogonalization process removes components of a vector that are parallel to a set of orthogonal vectors. Given a list of mutually orthogonal vectors {v1,v2,…,vk}\{\mathbf{v}_1, \mathbf{v}_2, \ldots, \mathbf{v}_k\}{v1​,v2​,…,vk​} and a new vector u\mathbf{u}u, the orthogonal component is computed as:

u⊥=u−∑i=1kvi⋅uvi⋅vivi \mathbf{u}_{\perp} = \mathbf{u} - \sum_{i=1}^{k} \frac{\mathbf{v}_i \cdot \mathbf{u}}{\mathbf{v}_i \cdot \mathbf{v}_i} \mathbf{v}_i u⊥​=u−i=1∑k​vi​⋅vi​vi​⋅u​vi​

Steps:

  1. Start with a copy of the new vector
  2. For each orthogonal vector vi\mathbf{v}_ivi​ in the list:
    • Compute the projection factor fi=vi⋅uvi⋅vif_i = \frac{\mathbf{v}_i \cdot \mathbf{u}}{\mathbf{v}_i \cdot \mathbf{v}_i}fi​=vi​⋅vi​vi​⋅u​
    • Subtract the projection: u←u−fivi\mathbf{u} \leftarrow \mathbf{u} - f_i \mathbf{v}_iu←u−fi​vi​
  3. Return the resulting orthogonal vector

Example

vecs={(10)},u=(34)⇒u⊥=(04) \text{vecs} = \left\{\begin{pmatrix} 1 \\ 0 \end{pmatrix}\right\}, \quad \mathbf{u} = \begin{pmatrix} 3 \\ 4 \end{pmatrix} \quad \Rightarrow \quad \mathbf{u}_{\perp} = \begin{pmatrix} 0 \\ 4 \end{pmatrix} vecs={(10​)},u=(34​)⇒u⊥​=(04​)
