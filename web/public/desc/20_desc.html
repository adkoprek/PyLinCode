ortho_base

Task

Create a function named ortho_base that takes a list of linearly independent vectors as its argument and returns a tuple of mutually orthogonal vectors that span the same space using the Gram-Schmidt process.

Input

  • vecs: list[vec] - List of linearly independent vectors

Output

  • tuple[vec] - Tuple of mutually orthogonal vectors

Theory

The Gram-Schmidt process converts a set of linearly independent vectors into a set of mutually orthogonal vectors that span the same space.

Steps:

  1. Keep the first vector unchanged: u1=v1\mathbf{u}_1 = \mathbf{v}_1u1​=v1​
  2. For each subsequent vector vi\mathbf{v}_ivi​ (where i=2,3,…,ni = 2, 3, \ldots, ni=2,3,…,n):
    • Use the ortho function to remove all components parallel to the previously computed orthogonal vectors {u1,u2,…,ui−1}\{\mathbf{u}_1, \mathbf{u}_2, \ldots, \mathbf{u}_{i-1}\}{u1​,u2​,…,ui−1​}
    • The result is ui\mathbf{u}_iui​, which is orthogonal to all previous vectors
  3. Return the tuple of orthogonal vectors (u1,u2,…,un)(\mathbf{u}_1, \mathbf{u}_2, \ldots, \mathbf{u}_n)(u1​,u2​,…,un​)

Example

Input: {(10),(11)}⇒Output: ((10),(01)) \text{Input: } \left\{\begin{pmatrix} 1 \\ 0 \end{pmatrix}, \begin{pmatrix} 1 \\ 1 \end{pmatrix}\right\} \quad \Rightarrow \quad \text{Output: } \left(\begin{pmatrix} 1 \\ 0 \end{pmatrix}, \begin{pmatrix} 0 \\ 1 \end{pmatrix}\right) Input: {(10​),(11​)}⇒Output: ((10​),(01​))

Test

vecs: list[vec] = [[1, 0], [1, 1]]
result = ortho_base(vecs)
