Create a function named \texttt{vec\_scl} that takes a vector and a scalar value and returns the scalar multiple of the vector.

\[
3 \cdot
\begin{pmatrix} 2 \\ -1 \\ 4 \end{pmatrix}
=
\begin{pmatrix} 6 \\ -3 \\ 12 \end{pmatrix}
\]

\begin{verbatim}
a: vec = [2, -1, 4]
assert vec_scl(a, 3) == [6, -3, 12]
\end{verbatim}

\begin{itemize}
    \item Test Cases: $50$
    \item Max Dimension: $10$
    \item Tolerance: $\,$
\end{itemize}
