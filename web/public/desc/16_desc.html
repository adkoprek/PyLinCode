solve

Task

Create a function named solve that takes a matrix and a vector as its arguments and solves the linear system Ax=bA\mathbf{x} = \mathbf{b}Ax=b using LU decomposition. If AAA is singular, your function must raise a SingularError. If the dimensions of AAA and b\mathbf{b}b are incompatible or AAA is not square, your function must raise a ShapeMismatchedError.

You must implement the following helper functions:

  • for_sub(L: mat, b: vec) -> vec - Performs forward substitution to solve Ly=bL\mathbf{y} = \mathbf{b}Ly=b
  • bck_sub(U: mat, y: vec) -> vec - Performs backward substitution to solve Ux=yU\mathbf{x} = \mathbf{y}Ux=y

Input

  • A: mat - Coefficient matrix (must be square)
  • b: vec - Right-hand side vector

Output

  • vec - Solution vector x\mathbf{x}x

Theory

To solve Ax=bA\mathbf{x} = \mathbf{b}Ax=b using LU decomposition:

Step 1: Verify that AAA is square. If not, raise ShapeMismatchedError.

Step 2: Compute the LU decomposition with partial pivoting: PA=LUPA = LUPA=LU using the lu function.

Step 3: Apply the permutation to the right-hand side: b′=Pb\mathbf{b}' = P\mathbf{b}b′=Pb using mat_vec_mul.

Step 4: Solve Ly=b′L\mathbf{y} = \mathbf{b}'Ly=b′ for y\mathbf{y}y using forward substitution (for_sub).

Step 5: Solve Ux=yU\mathbf{x} = \mathbf{y}Ux=y for x\mathbf{x}x using backward substitution (bck_sub). If a diagonal element of UUU is zero (within machine precision), raise SingularError.
