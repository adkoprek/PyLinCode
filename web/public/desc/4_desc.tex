Create a function named \texttt{vec\_len} that returns the Euclidean length of a vector.

\[
\left\| \begin{pmatrix} 3 \\ 4 \end{pmatrix} \right\| = \sqrt{3^2 + 4^2} = 5
\]

\begin{verbatim}
a: vec = [3, 4]
assert vec_len(a) == 5
\end{verbatim}

\begin{itemize}
    \item Test Cases: $50$
    \item Max Dimension: $10$
    \item Tolerance: $\,$
\end{itemize}
