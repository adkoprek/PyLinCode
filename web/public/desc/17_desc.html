inv

Task

Create a function named inv that takes a matrix as its argument and returns its inverse. If AAA is singular, your function must raise a SingularError. If AAA is not square, your function must raise a ShapeMismatchedError.

Input

  • a: mat - Input matrix (must be square and non-singular)

Output

  • mat - Inverse of the input matrix

Theory

To compute the matrix inverse A−1A^{-1}A−1 using LU decomposition:

Step 1: Verify that AAA is square. If not, raise ShapeMismatchedError.

Step 2: Compute the LU decomposition with partial pivoting: PA=LUPA = LUPA=LU using the lu function.

Step 3: Create the identity matrix III of size n×nn \times nn×n.

Step 4: For each column iii of the identity matrix:

  • Extract column ei\mathbf{e}_iei​ from III
  • Apply the permutation: ei′=Pei\mathbf{e}_i' = P\mathbf{e}_iei′​=Pei​
  • Solve Lyi=ei′L\mathbf{y}_i = \mathbf{e}_i'Lyi​=ei′​ using for_sub
  • Solve Uxi=yiU\mathbf{x}_i = \mathbf{y}_iUxi​=yi​ using bck_sub (will raise SingularError if matrix is singular)
  • Column xi\mathbf{x}_ixi​ becomes the iii-th column of A−1A^{-1}A−1

Step 5: Transpose the result to obtain A−1A^{-1}A−1.

Example

(2113)−1=(0.6−0.2−0.20.4) \begin{pmatrix} 2 & 1 \\ 1 & 3 \end{pmatrix}^{-1} = \begin{pmatrix} 0.6 & -0.2 \\ -0.2 & 0.4 \end{pmatrix} (21​13​)−1=(0.6−0.2​−0.20.4​)
