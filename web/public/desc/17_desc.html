vec_prj

Task

Create a function named vec_prj that takes two vectors as its arguments and returns the projection of vector b\mathbf{b}b onto vector a\mathbf{a}a.

Input

  • a: vec - Vector to project onto
  • b: vec - Vector to be projected

Output

  • vec - Projection of b\mathbf{b}b onto a\mathbf{a}a

Theory

The projection of vector b\mathbf{b}b onto vector a\mathbf{a}a is given by:
proja(b)=a⋅ba⋅aa \text{proj}_{\mathbf{a}}(\mathbf{b}) = \frac{\mathbf{a} \cdot \mathbf{b}}{\mathbf{a} \cdot \mathbf{a}} \mathbf{a} proja​(b)=a⋅aa⋅b​a

This formula computes the scalar factor a⋅ba⋅a\frac{\mathbf{a} \cdot \mathbf{b}}{\mathbf{a} \cdot \mathbf{a}}a⋅aa⋅b​ and multiplies it by vector a\mathbf{a}a to obtain the projection.

Example

proj(10)(34)=31(10)=(30) \text{proj}_{\begin{pmatrix} 1 \\ 0 \end{pmatrix}}\begin{pmatrix} 3 \\ 4 \end{pmatrix} = \frac{3}{1} \begin{pmatrix} 1 \\ 0 \end{pmatrix} = \begin{pmatrix} 3 \\ 0 \end{pmatrix} proj(10​)​(34​)=13​(10​)=(30​)

Test

b: vec = [3, 4]
