mat_row

Task

Create a function named mat_row that takes a matrix and an index as its arguments and returns the specified row of the matrix (zero-indexed).

Input

  • A: mat - Input matrix
  • i: int - Row index (zero-indexed)

Output

  • vec - The iii-th row of the matrix

Example

(1234)→(12) (0th row) \begin{pmatrix} 1 & 2 \\ 3 & 4 \end{pmatrix} \rightarrow \begin{pmatrix} 1 & 2 \end{pmatrix} \text{ (0th row)} (13​24​)→(1​2​) (0th row)

Test

A: mat = [[1, 2, 3], [4, 5, 6]]
assert mat_row(A, 1) == [4, 5, 6]

Cases

  • Dimension: 111 to 101010 (rows and columns)
