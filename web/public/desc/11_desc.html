mat_ide

Task

Create a function named mat_ide that takes a size as its argument and returns the identity matrix of size n×nn \times nn×n.

Input

  • n: int - Size of the identity matrix

Output

  • mat - Identity matrix of size n×nn \times nn×n

Example

I3=(100010001) I_3 = \begin{pmatrix} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 1 \end{pmatrix} I3​=⎝⎛​100​010​001​⎠⎞​

Test

assert mat_ide(3) == [[1, 0, 0], [0, 1, 0], [0, 0, 1]]

Cases

  • Dimension: 111 to 101010 (rows and columns)
