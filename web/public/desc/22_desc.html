det

Task

Create a function named det that takes a matrix as its argument and returns its determinant using LU decomposition. If the matrix is not square, your function must raise a ShapeMismatchedError.

Input

  • a: mat - Input matrix (must be square)

Output

  • float - Determinant of the input matrix

Theory

The determinant can be efficiently computed using LU decomposition with partial pivoting. Given PA=LUPA = LUPA=LU, the determinant is:

det⁡(A)=det⁡(P−1LU)=det⁡(P−1)⋅det⁡(L)⋅det⁡(U) \det(A) = \det(P^{-1}LU) = \det(P^{-1}) \cdot \det(L) \cdot \det(U) det(A)=det(P−1LU)=det(P−1)⋅det(L)⋅det(U)

Since:

  • det⁡(L)=1\det(L) = 1det(L)=1 (L has ones on the diagonal)
  • det⁡(U)=∏i=1nUii\det(U) = \prod_{i=1}^{n} U_{ii}det(U)=∏i=1n​Uii​ (product of diagonal elements)
  • det⁡(P−1)=det⁡(PT)=det⁡(P)=(−1)number of swaps\det(P^{-1}) = \det(P^T) = \det(P) = (-1)^{\text{number of swaps}}det(P−1)=det(PT)=det(P)=(−1)number of swaps

  1. Verify that AAA is square. If not, raise ShapeMismatchedError.

  3. Compute det⁡(U)\det(U)det(U) as the product of diagonal elements: ∏i=0n−1Uii\prod_{i=0}^{n-1} U_{ii}∏i=0n−1​Uii​
  4. Determine the sign from the permutation matrix PPP:

Computing the Sign of P:

The sign of a permutation equals (−1)number of transpositions(-1)^{\text{number of transpositions}}(−1)number of transpositions. We can compute this using cycle decomposition:

  • Convert the permutation matrix to a permutation array: for each row iii, find which column contains the 1, giving p[i]p[i]p[i]
  • Count the number of disjoint cycles in the permutation:
    • Start with all positions unvisited
    • For each unvisited position iii:
      • Increment cycle count
      • Follow the chain: i→p[i]→p[p[i]]→…i \to p[i] \to p[p[i]] \to \ldotsi→p[i]→p[p[i]]→… until returning to iii
      • Mark all positions in this cycle as visited
  • The number of transpositions = n−number of cyclesn - \text{number of cycles}n−number of cycles
  • Sign = (−1)n−cycles(-1)^{n - \text{cycles}}(−1)n−cycles

Example: For P=(010001100)P = \begin{pmatrix} 0 & 1 & 0 \\ 0 & 0 & 1 \\ 1 & 0 & 0 \end{pmatrix}P=⎝⎛​001​100​010​⎠⎞​, the permutation is p=[1,2,0]p = [1, 2, 0]p=[1,2,0]. This forms one cycle: 0→1→2→00 \to 1 \to 2 \to 00→1→2→0. So cycles = 1, and sign = (−1)3−1=1(-1)^{3-1} = 1(−1)3−1=1.

  1. Return: det⁡(A)=sign×det⁡(U)\det(A) = \text{sign} \times \det(U)det(A)=sign×det(U)
