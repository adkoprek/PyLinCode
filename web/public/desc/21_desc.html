qr

Task

Create a function named qr that takes a matrix as its argument and returns its QR decomposition using the Householder reflection method. The function returns a tuple of two matrices: Q (orthogonal matrix) and R (upper triangular matrix), such that A=QRA = QRA=QR.

Input

  • a: mat - Input matrix

Output

  • tuple[mat, mat] - Tuple containing (Q, R) matrices

Theory

The QR decomposition factors a matrix AAA into the product of an orthogonal matrix QQQ and an upper triangular matrix RRR, such that A=QRA = QRA=QR. The Householder reflection method achieves this by applying a sequence of orthogonal transformations to progressively zero out elements below the diagonal.

Householder Reflection Steps:

For each column iii (from 0 to min⁡(m−1,n)\min(m-1, n)min(m−1,n)):

  1. Extract the subvector x\mathbf{x}x from column iii starting at row iii
  2. Compute its length: ∥x∥\|\mathbf{x}\|∥x∥
  3. Determine the sign for numerical stability and compute:
    α=−sign(x0)⋅∥x∥\alpha = -\text{sign}(x_0) \cdot \|\mathbf{x}\|α=−sign(x0​)⋅∥x∥
  4. Create the reflection vector:
    u=x−αe1\mathbf{u} = \mathbf{x} - \alpha \mathbf{e}_1u=x−αe1​
    where e1=(1,0,0,…)\mathbf{e}_1 = (1, 0, 0, \ldots)e1​=(1,0,0,…)
  5. Normalize: v=u∥u∥\mathbf{v} = \frac{\mathbf{u}}{\|\mathbf{u}\|}v=∥u∥u​
  6. Construct the Householder matrix:
    H=I−2vvTH = I - 2\mathbf{v}\mathbf{v}^TH=I−2vvT
  7. Update: R←HRR \leftarrow HRR←HR and Q←QHTQ \leftarrow QH^TQ←QHT

The resulting QQQ is orthogonal (QTQ=IQ^TQ = IQTQ=I) and RRR is upper triangular.

Test

a: mat = [[1, 1], [1, 0]]
Q, R = qr(a)
m, n = len(a), len(a[0])
assert Q.shape == (m, m), f"Q must be shape ({m}, {m})"
assert R.shape == (m, n), f"R must be shape ({m}, {n})"
np.testing.assert_allclose(Q @ Q.T, np.eye(m), atol=1e-10)
np.testing.assert_allclose(Q @ R, a, atol=1e-10)
assert np.allclose(R, np.triu(R)), "R is not upper triangular"
